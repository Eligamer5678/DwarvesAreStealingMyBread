# Dwarves are stealing my bread
The BREAD rules above all dwarves. All dwarves pray to the BREAD. Dwarves must recreate the BREAD.


#### Infomation
Contributers are very much welcome, feel free to draw assets & help code this game.
- This uses a custom framework I have crafted called The Dicey Game Engine.
- It feels similar to godot using dyamic scene loading & signals, just using js canvas layers & classes/objects instead of nodes.
- See scenes/main.js
